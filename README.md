В процессе разработки данной программы были созданы различные функции для работы с данными о книгах. Каждая функция выполняет определенную задачу, такую как добавление книги, удаление книги, редактирование данных о книге, просмотр всех данных, поиск книги по названию, сортировка списка книг по различным критериям и другие. Для улучшения пользовательского опыта была реализована возможность продолжения работы после выполнения каждой операции. Пользователь может выбирать, хочет ли он продолжить работу с программой или завершить ее. В целом, данная программа предоставляет базовый набор функций для управления списком книг.
